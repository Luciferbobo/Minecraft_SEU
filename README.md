# Minecraft_SEU

东南大学我的世界Minecraft建造存档

由SEU·Minecraft团队建造

最后施工日期2020.9.13，现已移交国建复现SEU

已完成设施：SEU整体位置构图、李文正图书馆、桃园操场(框架)、桃园篮球场、钢菊体育场(框架)
